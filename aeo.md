# AEO

## Scope

This file covers answer-engine-oriented practices from the report.

## Rules

### 1) Use headings that express the question or answer clearly
- **Rule name:** Question-led headings
- **Description:** Structure sections so a user question can map directly to the heading and answer.
- **Source URL:** https://www.lumar.io/blog/best-practice/4-pillar-geo-strategy-framework-for-ai-search-visibility/
- **Source organization:** Lumar
- **Source type:** Industry analysis
- **Classification:** Emerging GEO Practice
- **Severity:** Opportunity
- **Confidence level:** Medium
- **Evidence strength:** Moderate
- **Why it matters:** It may make extraction easier for answer engines.
- **Expected impact:** Better answer visibility potential.
- **Implementation guidance:** Use `H2/H3` headings phrased as the question or topic.
- **Example:** `How does canonicalization work?`
- **Exceptions:** Not every page needs a Q&A format.
- **Potential drawbacks:** Overuse can make pages repetitive.
- **Conflicting opinions:** Google does not prescribe this as a formal ranking rule.
- **Notes:** This is part of the report’s answer-first guidance.

### 2) Keep concise definitions near the top when the page is definitional
- **Rule name:** Concise definition
- **Description:** For definition-style pages, provide a short direct explanation early in the page.
- **Source URL:** https://www.lumar.io/blog/best-practice/4-pillar-geo-strategy-framework-for-ai-search-visibility/
- **Source organization:** Lumar
- **Source type:** Industry analysis
- **Classification:** Experimental
- **Severity:** Opportunity
- **Confidence level:** Low
- **Evidence strength:** Limited
- **Why it matters:** The report says it may help AI snippets, but evidence is not strong.
- **Expected impact:** Possible snippet usefulness.
- **Implementation guidance:** Keep the definition short, accurate, and useful.
- **Example:** A one-paragraph definition at the start of an explainer page.
- **Exceptions:** Landing pages and commercial pages may not use this format.
- **Potential drawbacks:** Can add fluff if not genuinely informative.
- **Conflicting opinions:** None in the report.
- **Notes:** Mark as experimental.
